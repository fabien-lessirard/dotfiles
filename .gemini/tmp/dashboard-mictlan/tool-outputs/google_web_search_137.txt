To programmatically check your Firebase project's plan (Spark or Blaze), you need to determine the billing status of its associated Google Cloud project. Firebase projects are intrinsically linked to Google Cloud projects, and the presence of an active Cloud Billing account on the Google Cloud project indicates a Firebase Blaze plan. Conversely, if no billing account is linked, the Firebase project is on the Spark plan.[1][2]

You can check the billing status using either the `gcloud` Command-Line Interface (CLI) or the Google Cloud Billing API.

### Using the `gcloud` CLI

The `gcloud` CLI provides a straightforward way to check a project's billing status.

1.  **Identify your Google Cloud Project ID:**
    Your Firebase project's ID is the same as its underlying Google Cloud Project ID. You can find this in the Firebase console under Project settings, or by running `firebase projects:list` if you have the Firebase CLI installed.[1]

2.  **Check billing status:**
    Execute the following `gcloud` command, replacing `YOUR_PROJECT_ID` with your actual project ID:

    ```bash
    gcloud billing projects describe YOUR_PROJECT_ID
    ```

    This command will return information about the project's billing. Look for the `billingEnabled` field in the output.[3]

    *   If `billingEnabled` is `True`, your Firebase project is on the **Blaze plan**.[4]
    *   If `billingEnabled` is `False`, your Firebase project is on the **Spark plan**.[4]

    **Example Output (Blaze Plan):**
    ```
    billingAccountName: billingAccounts/012345-6789AB-CDEF01
    billingEnabled: true
    name: projects/YOUR_PROJECT_ID/billingInfo
    projectId: YOUR_PROJECT_ID
    ```

    **Example Output (Spark Plan):**
    ```
    billingEnabled: false
    name: projects/YOUR_PROJECT_ID/billingInfo
    projectId: YOUR_PROJECT_ID
    ```

### Using the Google Cloud Billing API

For more complex programmatic integrations, you can use the Google Cloud Billing API.

1.  **Enable the Cloud Billing API:**
    Ensure the Cloud Billing API is enabled for your Google Cloud project.

2.  **Make an API call:**
    Use the `projects.getBillingInfo` method of the Cloud Billing API. This method returns a `ProjectBillingInfo` resource, which includes the `billingEnabled` field.[4][5]

    You will need to authenticate your API requests, typically using a service account with appropriate permissions (e.g., `billing.projects.get`).

    **Example (using `curl` for demonstration):**
    ```bash
    curl -H "Authorization: Bearer $(gcloud auth print-access-token)" \
         "https://cloudbilling.googleapis.com/v1/projects/YOUR_PROJECT_ID/billingInfo"
    ```

    The JSON response will contain the `billingEnabled` field, indicating the project's billing status.[4]

    *   If `"billingEnabled": true`, your Firebase project is on the **Blaze plan**.
    *   If `"billingEnabled": false`, your Firebase project is on the **Spark plan**.

By checking the `billingEnabled` status of your Google Cloud project, you can programmatically determine whether your Firebase project is on the Spark or Blaze plan.
